# Project Skills

This folder contains project-local Codex skills for TradingAgents-CN.

## Available Skills

| Skill | Use When |
| --- | --- |
| `design-thinking-partner` | Discuss architecture options, workflow changes, module boundaries, and design tradeoffs before implementation. |
| `implementation-plan-documentation` | Create or update feature implementation plans, rollout steps, dependencies, and acceptance checks. |
| `module-usage-documentation` | Write practical usage guides for project modules, tools, and workflows. |
| `analysis-documentation` | Explain how existing source code, services, or architecture currently works. |
| `temp-work-documentation` | Keep temporary task notes or working agreements. |
| `generate-commit-comment` | Summarize current repository changes into commit comments. |

For fund-analysis expansion work, use `design-thinking-partner` for discussion and `implementation-plan-documentation` for the implementation plan.
